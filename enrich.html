Passive DNS

Database

Passive DNS (pDNS) is a system of recording information about which domain names (DNS records) are resolved to which IP addresses without actively querying DNS servers for current data. Unlike traditional DNS queries, which actively ask DNS servers to resolve a domain name at the moment of the query, passive DNS collects data by monitoring the DNS traffic as it occurs naturally within networks. This method allows for the collection of DNS resolution data over time, without sending any additional queries that could alert malicious actors or affect the performance of the DNS infrastructure.
Data leaks

Database

Data Leaks database is a repository that aggregates, indexes, and organizes information from data breaches and leaks. The database contains records that have been exposed through various cybersecurity attacks, published insider leakes, and inadvertent disclosures. The information in these databases includes personal data (such as names, email addresses, compromised credentials), corporate data, and more.
Indicators of compromise

Database

Database of Indicators of Compromise (IOCs) is a collection of digital artifacts that are associated with malicious cyber activity. IOCs serve as evidence that a network or system may have been compromised. These artifacts include a wide range of data points, mainly focused on malware system and network activity.
